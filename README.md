# 🛒 QA E-Commerce Testing Project (JPetStore)

## 📌 Descripción
Proyecto de testing manual sobre una aplicación e-commerce (JPetStore), simulando un entorno real de trabajo QA.

---

## 🔍 Alcance de pruebas

Se realizaron pruebas sobre el flujo completo de compra:

- Login de usuario  
- Navegación de productos  
- Visualización de detalle  
- Agregado al carrito  
- Checkout  
- Confirmación de compra  

---

## 🧪 Tipos de pruebas realizadas

- Testing funcional  
- Testing de regresión  
- Pruebas de integración  

---

## ✅ Casos de prueba

Se diseñaron y ejecutaron casos de prueba incluyendo:

- Escenarios positivos  
- Escenarios negativos  
- Resultados esperados vs obtenidos  
Los casos de prueba pueden visualizarse en el archivo:
📄 Casos de Prueba Cintia Diaz.xlsx
---

## 🐞 Bugs detectados

- Problemas en validaciones de formularios  
- Comportamientos inesperados en el flujo de compra  
- Manejo incorrecto de errores  

---

## 🛠️ Herramientas utilizadas

- Navegador web  
- Excel (casos de prueba)  
- Jira (seguimiento de bugs)

---

## 🎯 Objetivo

Aplicar buenas prácticas de QA en pruebas de una aplicación e-commerce simulando un entorno laboral real.
